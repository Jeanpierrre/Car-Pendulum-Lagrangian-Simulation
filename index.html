Simulación Lagrangiana: carrito–péndulo en plano inclinado

El movimiento se calcula con las ecuaciones en x y θ. Para dibujar el péndulo se aplica la relación angular solicitada: θ = φ − α, por lo tanto φ = θ + α.

Modelo matemático

2 GDL: x, θ
Lagrangiano
L = T − V
T = ½(mC + mB)ẋ² + ½mBl²θ̇² + mBlẋθ̇cosθ
V = (mC + mB)gxsinα − mBglcosθ + ½kx²
Ecuaciones de movimiento
(mC + mB)ẍ + mBl(θ̈cosθ − θ̇²sinθ) + kx + (mC + mB)gsinα = 0
mBl²θ̈ + mBlẍcosθ + mBglsinθ = 0
Forma integrada
ẍ = [mBgsinθcosθ + mBlθ̇²sinθ − kx − (mC + mB)gsinα] / [mC + mBsin²θ]
θ̈ = −(ẍcosθ + gsinθ)/l
Ángulo visual del péndulo
θ = φ − α   ⇒   φ = θ + α
